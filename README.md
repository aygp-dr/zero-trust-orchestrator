# zero-trust-orchestrator

Security policy dashboard. Terminal UI built with [Bubble Tea](https://github.com/charmbracelet/bubbletea).

## Run

```bash
go run .
```

## Build

```bash
make build
./bin/zero-trust-orchestrator
```

## Controls

- `j/k`: navigate
- `Enter`: select/details
- `q`: quit
